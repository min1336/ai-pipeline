**제목:** 블로그 검색에서 한글 검색어가 0건 반환됨

**본문:**
### 재현 방법
1. /blog 페이지 접속
2. 검색창에 "TypeScript" 입력 → 정상 (결과 있음)
3. 검색창에 "타입스크립트" 입력 → 0건 반환

### 기대 동작
한글 검색어도 제목/요약에서 매칭되어야 함

### 실제 동작
한글 검색어 입력 시 항상 빈 결과 반환

### 에러 메시지
콘솔에 에러 없음. API 응답: `{ data: [], total: 0 }`

### 환경
- Chrome 125, macOS 15.2
- 프로덕션 환경 (blog-ebon-eta-50.vercel.app)

### 발생 빈도
항상 재현됨
