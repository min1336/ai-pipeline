# Phase 2: 코드 품질 검증

## 보안 (25점)

| 항목 | 감점 |
|------|------|
| SQL injection 가능성 | -25 (즉시 block) |
| XSS 가능성 | -25 (즉시 block) |
| 하드코딩된 비밀값 | -25 (즉시 block) |
| CSRF 보호 누락 | -10 |
| 입력 검증 누락 (시스템 경계) | -5 |
| 안전하지 않은 역직렬화 | -10 |
| 인증/인가 우회 가능성 | -25 (즉시 block) |

## 테스트 (25점)

| 항목 | 감점 |
|------|------|
| 새 기능에 테스트 없음 | -15 |
| 엣지 케이스 미처리 | -5 |
| 기존 테스트 깨짐 | -25 (즉시 block) |
| 테스트가 구현을 검증하지 않음 | -10 |

|------|------|
| 프로젝트 컨벤션 미준수 | -5 |

| 과도한 복잡도 (불필요한 추상화) | -5 |
| 에러 핸들링 누락 (시스템 경계) | -5 |
| 불필요한 코드 변경 | -5 |

| 항목 | 감점 |
|------|------|

| 불필요한 동기 처리 | -5 |
| 큰 파일 전체 로드 | -5 |

| 잘못된 주석/문서 | -5 |
